# Optional Directives

These pair with the opt-in skills in `dot-claude/optional-skills/`. If you install one of those skills, its matching directive here becomes active; until then these files are inert.

- `cre_deal_notes.md` pairs with the `teardown-cre-deal` skill (commercial real estate deal analysis)
- `recruiting_prep.md` pairs with the `prep-recruiting-conversation` skill (job and internship recruiting)
