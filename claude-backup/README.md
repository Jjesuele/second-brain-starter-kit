# Claude Backup

This folder holds a copy of your personal `~/.claude/CLAUDE.md` (your global Claude operating file), which lives outside this vault and would otherwise have no backup.

The setup directive puts the first copy here. After that, whenever your `~/.claude/CLAUDE.md` changes, refresh the copy here so git keeps its history. The monthly audit directive includes this check.

Your installed Claude skills do not need a mirror here; they are identical to `dot-claude/skills/`, which is already in this repo.
