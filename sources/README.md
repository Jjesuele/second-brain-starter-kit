# Sources

Raw material waiting to be turned into brain notes: transcripts, document exports, syllabi, meeting notes, anything worth mining later.

Make subfolders per area as material arrives (for example `school-docs/`, `work-docs/`). When a folder grows past a handful of files, add an `INDEX.md` inside it with one line per file so the AI can find things without opening everything.

Nothing in here is a finished note. When something gets mined into `brain/`, the brain note cites the source file here.
